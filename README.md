## Инструкции по сборке и запуску сервиса
Для сборки и запуска вашего Spring Boot приложения вам понадобятся следующие шаги:
- Откройте терминал в корневой директории вашего проекта и выполните следующую команду для сборки вашего приложения:
./gradlew build
Эта команда соберет ваш проект и создаст исполняемый JAR-файл в директории build/libs.
- После успешной сборки вы можете запустить приложение, используя следующую команду:
java -jar build/libs/library-service-0.0.1-SNAPSHOT.jar
## Либо вы можете запустить приложение через Docker
- Выполните следующую команду в терминале, находясь в корневой директории вашего проекта, чтобы собрать Docker-образ вашего приложения:
docker build -t library-service .
- После успешной сборки образа запустите контейнер с вашим приложением, используя следующую команду:
docker run -p 8080:8080 library-service
